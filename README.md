# DeFiPilot 

**DeFiPilot** is an AI-powered DeFi yield aggregator dashboard that helps users discover the best yield farming opportunities, simulate investments, and track their crypto portfolio.

Built using **React, Spring Boot, and Starknet ecosystem tools**.

---

##  Live Demo

Frontend: https://defipilot-starkzap-frontend-xboe.vercel.app/
Backend API: https://defipilot-starkzap-backend-y6pf.onrender.com

---

##  Features

*  Wallet connection (MetaMask)
*  Portfolio tracker
*  AI pool recommendation
*  Yield analytics dashboard
*  Investment simulator
*  DeFi pool explorer
*  Crypto price charts
*  Withdraw with transaction notification

---

## AI Recommendation

The AI module analyzes:

* APR
* Risk score
* Liquidity strength

And recommends the **best yield pool for investment**.

##  Installation

Clone the repository:

git clone https://github.com/VrindaSuryavanshi/defipilot-starkzap

### Frontend

cd frontend
npm install
npm start

### Backend

cd backend
mvn spring-boot:run
